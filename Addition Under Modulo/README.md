# Modular Addition

## Problem Statement

Given three integers:

```text
a, b, M
```

Compute:

```text
(a + b) mod M
```

---

# Examples

### Example 1

```text
Input:
a = 10
b = 20
M = 3

Output:
0
```

### Explanation

```text
(10 + 20) % 3
= 30 % 3
= 0
```

---

### Example 2

```text
Input:
a = 100
b = 13
M = 107

Output:
6
```

### Explanation

```text
(100 + 13) % 107
= 113 % 107
= 6
```

---

## Constraints:

- 1 ≤ a, b , M ≤ 10^9
