# Absolute Value of an Integer

## Problem Statement

Given an integer `n`, return its absolute value.

The absolute value of a number is its distance from `0`.

---

# Examples

### Example 1

```text
Input:
n = -32

Output:
32
```

### Explanation

```text
Absolute value of -32 is 32
```

---

### Example 2

```text
Input:
n = 45

Output:
45
```

### Explanation

```text
Positive numbers remain unchanged
```

---

## Constraints:

- -10^6 ≤ n ≤ 10^6
