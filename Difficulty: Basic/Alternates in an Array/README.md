# Print Alternate Elements of an Array

## Problem Statement

Given an array `arr[]`, return elements in **alternate order starting from index 0**.

👉 That means:
```
Take index 0, skip 1, take 2, skip 3, ...
```

---

# Examples

### Example 1

**Input**
```
arr = [1, 2, 3, 4]
```

**Output**
```
1 3
```

---

### Example 2

**Input**
```
arr = [1, 2, 3, 4, 5]
```

**Output**
```
1 3 5
```

---

## Constraints:

- 1 <=  arr.size <= 10^5
- 1 <= arr[i] <= 10^5
