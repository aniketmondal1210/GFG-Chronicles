# Check if String Can Be Rearranged to Palindrome

## Problem Statement

Given a string `s`, determine whether its characters can be rearranged to form a palindrome.

Return:
```
true  → if possible
false → otherwise
```

---

# Examples

### Example 1

**Input**
```
s = "baba"
```

**Output**
```
true
```

---

### Example 2

**Input**
```
s = "geeksogeeks"
```

**Output**
```
true
```

---

### Example 3

**Input**
```
s = "geeksforgeeks"
```

**Output**
```
false
```

---

## Constraints:

- 1 ≤ s.length ≤ 10^6
- s consists of only lowercase English letters.
