# Binary Addition with 1

## Problem Statement
You are given a **binary string `s`** of length **n**.  
Your task is to perform **binary addition of the string with `1`** and return the resulting binary string.

---

## Examples

### Example 1
**Input:**

n = 4

s = "1010"


**Output:**

1011


**Explanation:**
- Decimal value of `1010` is **10**
- `10 + 1 = 11`
- Binary representation of `11` is **1011**

---

### Example 2
**Input:**

n = 3

s = "111"


**Output:**

1000


**Explanation:**
- Decimal value of `111` is **7**
- `7 + 1 = 8`
- Binary representation of `8` is **1000**

---

## Expected Complexity
- **Time Complexity:** `O(n)`
- **Auxiliary Space:** `O(1)`

---

## Constraints

- 1 ≤ n ≤ 10000
