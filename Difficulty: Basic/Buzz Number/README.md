# Buzz Number

## Problem Statement

A number `n` is called a **Buzz Number** if:

- It ends with `7`
OR
- It is divisible by `7`

Return:

- `true` → if `n` is a Buzz Number
- `false` → otherwise

---

# Examples

### Example 1

```text
Input:
n = 63

Output:
true
```

### Explanation

```text
63 is divisible by 7

63 % 7 == 0
```

So it is a Buzz Number.

---

### Example 2

```text
Input:
n = 72

Output:
false
```

### Explanation

```text
72 is not divisible by 7
72 does not end with 7
```

So it is not a Buzz Number.

---

## Constraints :

- 0 <= n <= 10^9
