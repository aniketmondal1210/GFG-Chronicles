# Reverse and Swap Two Numbers

## Problem Statement
Rahul gives Ankush two integers `a` and `b`.  
The task is:

1. **Reverse the digits** of both numbers.  
2. **Swap the reversed results**.  
3. Print the final swapped reversed numbers.

---

## Examples

### Example 1
**Input:**  

a = 1234, b = 67890


**Process:**  
- Reverse `a`: 1234 → 4321  
- Reverse `b`: 67890 → 09876 → 9876  
- Swap them → Output: `9876 4321`

**Output:**  

9876 4321


---

### Example 2
**Input:**  

a = 10000, b = 3254


**Process:**  
- Reverse `a`: 10000 → 00001 → 1  
- Reverse `b`: 3254 → 4523  
- Swap them → Output: `4523 1`

**Output:**  

4523 1


---

## Constraints

- 0 ≤ a, b ≤ 100000
