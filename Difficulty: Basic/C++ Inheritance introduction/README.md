# Implement Cuboid and CuboidVol Classes

## Requirements

### Class 1: **Cuboid**
- Contains three integer fields:  
  `length`, `width`, `height`
- Has a method **display()** that prints:

length width height


---

### Class 2: **CuboidVol** (inherits from Cuboid)
- Has method **read_input()** → reads `length`, `width`, `height`
- Overrides **display()** method to print:

volume = length * width * height


---

## Example

### Input

12 10 2


### Output

12 10 2

240


Explanation:  
Volume = 12 × 10 × 2 = **240**

---

## Constraints

0 ≤ length, width, height ≤ 100
