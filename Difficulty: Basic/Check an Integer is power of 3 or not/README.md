# Check if a Number is a Power of Three

## Problem
Given a positive integer `N`, determine if it is a power of 3.

---

## Examples

### Example 1
**Input:**

N = 3

**Output:**

Yes

**Explanation:**  
3¹ = 3 → It is a power of 3.

---

### Example 2
**Input:**

N = 5

**Output:**

No

**Explanation:**  
No power of 3 equals 5.

---

## Constraints
- 1 ≤ N ≤ (32-bit largest integer)

---

## Expected Complexity
- **Time Complexity:** O(1)
- **Space Complexity:** O(1)
