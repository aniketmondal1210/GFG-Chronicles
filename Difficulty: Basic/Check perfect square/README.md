# Check if a Number is a Perfect Square (Using Addition & Subtraction)

## Problem
Given a positive integer **N**, check if it is a **perfect square** or not.  

⚡ Constraint: Try to solve this using only **addition** and **subtraction** (not multiplication, division, or square root functions).

---

## Examples

### Example 1
**Input:**

N = 35

**Output:**

0

**Explanation:**  
35 is not a perfect square.

---

### Example 2
**Input:**

N = 49

**Output:**

1

**Explanation:**  
49 is a perfect square (7 × 7).

---

## Constraints
- 1 ≤ N ≤ 100,000
