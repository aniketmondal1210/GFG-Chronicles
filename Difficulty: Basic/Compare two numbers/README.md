# Compare Two Numbers

## Problem Statement
You are given two numbers **a** and **b**.  
If **a > b**, print **1**.  
Otherwise, print **0**.

---

## Examples

### Example 1
**Input:**  

a = 4, b = 6

**Output:**  

0

**Explanation:**  
a is smaller than b.

---

### Example 2
**Input:**  

a = 2, b = -9

**Output:**  

1

**Explanation:**  
a is greater than b.

---

### Example 3
**Input:**  

a = 5, b = 5

**Output:**  

0

**Explanation:**  
a is not greater than b, both are equal.

---

## Constraints
-1000 ≤ a, b ≤ 1000
