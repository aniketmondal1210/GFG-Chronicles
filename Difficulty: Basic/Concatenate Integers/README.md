# Concatenate Two Integers

## Problem Statement

Given two integers `a` and `b`, you need to concatenate them so the output is `ab` and print it.

---

## Examples

### Example 1:
**Input:**  
`a = 5, b = 6`  
**Output:**  
`56`  
**Explanation:**  
Concatenating 5 and 6 gives 56.

### Example 2:
**Input:**  
`a = 1, b = 9`  
**Output:**  
`19`  
**Explanation:**  
Concatenating 1 and 9 gives 19.

---

## Constraints

- `0 ≤ a, b ≤ 10⁶`
