# Count Distinct Vowels

## Problem

Given a string `s` consisting of lowercase English letters, count the number of **distinct vowels** present in the string.

The vowels are:

```text
a, e, i, o, u
```

Return the count of unique vowels that appear at least once.

---

## Examples

### Example 1

**Input**

```text
s = "geeksforgeeks"
```

**Output**

```text
2
```

**Explanation**

Distinct vowels present:

```text
e, o
```

Count = `2`.

---

### Example 2

**Input**

```text
s = "world"
```

**Output**

```text
1
```

**Explanation**

Distinct vowels present:

```text
o
```

Count = `1`.

---

## Constraints:

- 1 ≤ |s| ≤ 10^4
