# Count Elements Less Than or Equal to X

Given an **unsorted array** `arr[]` and an integer `x`, find the **count of elements** in the array that are **less than or equal to `x`**.

---

## Examples

### Example 1

Input:
x = 9
arr = [10, 1, 2, 8, 4, 5]

Output:
5


**Explanation:**  
Elements less than or equal to `9` are: `1, 2, 8, 4, 5`

---

### Example 2

Input:
x = 2
arr = [1, 2, 2, 5, 7, 2, 9]

Output:
4


**Explanation:**  
Elements less than or equal to `2` are: `1, 2, 2, 2`
