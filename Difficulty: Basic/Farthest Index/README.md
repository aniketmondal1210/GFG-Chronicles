# Find the Farthest Index of x in an Array

Given:
- A positive integer `x`
- An array `arr` of positive integers  
Return the **last (farthest) index** where `x` occurs.  
If `x` is not present, return `-1`.

---

## Examples

### Example 1

Input: arr = [7, 42, 5, 6, 42, 8, 7, 5, 3, 6, 7], x = 7

Output: 11

Explanation: The last index containing value 7 is index 11.


### Example 2

Input: arr = [1, 2, 2], x = 2

Output: 1

Explanation: The farthest index containing value x is 1.


---

### Constraints

- 1 ≤ arr.size() ≤ 10^6
- 1 ≤ arr[i], x ≤ 10^5
