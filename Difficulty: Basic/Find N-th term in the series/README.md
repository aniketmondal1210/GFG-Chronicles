# Find the n-th Term of the Series (9, 33, 73, 129, ...)

## Problem Statement
You are given a series:

9, 33, 73, 129, ...

Your task is to find the **n-th term** of this series.

---

## Examples

### Example 1:
**Input:**  
`n = 4`

**Output:**  
`129`

**Explanation:**  
The series is: `9, 33, 73, 129, ...`  
The 4th term is `129`.

---

### Example 2:
**Input:**  
`n = 5`

**Output:**  
`201`

**Explanation:**  
The series is: `9, 33, 73, 129, 201, ...`  
The 5th term is `201`.

---

## Constraints
- `1 ≤ n ≤ 10^4`
