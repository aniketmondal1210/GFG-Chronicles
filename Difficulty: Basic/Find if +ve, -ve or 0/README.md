# Check if a Number is Positive, Negative, or Zero

## Problem
You are given a number `a`.  
Your task is to:  
- Print `"Positive"` if the number is positive  
- Print `"Negative"` if the number is negative  
- Print `"Zero"` if it is `0`  

---

## Examples

### Example 1
**Input:**

a = 4

**Output:**

Positive


---

### Example 2
**Input:**

a = -2

**Output:**

Negative


---

### Example 3
**Input:**

a = 0

**Output:**

Zero


---

## Constraints
- -10⁵ ≤ a ≤ 10⁵
