# Generate First N Fibonacci Numbers

## Problem
Given a number `n`, return an array containing the **first n Fibonacci numbers**.  

ğŸ”¹ The first two numbers of the Fibonacci series are `0` and `1`.

---

## Constraints
- `1 <= n <= 30`

---

## Examples

### Example 1
**Input**

n = 5

**Output**

[0, 1, 1, 2, 3]


---

### Example 2
**Input**

n = 7

**Output**

[0, 1, 1, 2, 3, 5, 8]


---

### Example 3
**Input**

n = 2

**Output**

[0, 1]
