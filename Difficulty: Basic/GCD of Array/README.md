# GCD of an Array

## Problem Statement
You are given an array of `n` positive integers.  
Your task is to find the **Greatest Common Divisor (GCD)** of all the elements in the array.

---

## Examples

### Example 1
**Input:**

n = 3
arr = [1, 2, 3]

**Output:**

1

**Explanation:**  
The GCD of 1, 2, and 3 is `1`.

---

### Example 2
**Input:**

n = 4
arr = [2, 4, 6, 8]

**Output:**

2

**Explanation:**  
The GCD of 2, 4, 6, and 8 is `2`.

---

## Constraints
- 1 ≤ N, arr[i] ≤ 10^5
