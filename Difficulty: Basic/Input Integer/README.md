# Add 10 to Input Number

## Problem Statement

Given a number `n`, take input and add `10` to its value, then print the result.

---

## Examples

### Example 1:
**Input:**  
n = 2  
**Output:**  
12  
**Explanation:**  
Input is 2; adding 10 gives 12.

### Example 2:
**Input:**  
n = 22  
**Output:**  
32  
**Explanation:**  
Input is 22; adding 10 gives 32.

---

## Constraints

- `-10⁶ ≤ n ≤ 10⁶`
