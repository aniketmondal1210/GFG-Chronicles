# Insert Element in Middle of Linked List

## Problem Statement

Given the head of a **singly linked list** and a value `x`, insert `x` **after the middle node** of the linked list.

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 4 , x = 3
```

Middle = `2`

**Output**
```
1 -> 2 -> 3 -> 4
```

---

### Example 2

**Input**
```
10 -> 20 -> 40 -> 50 , x = 30
```

Middle = `20`

**Output**
```
10 -> 20 -> 30 -> 40 -> 50
```

---

## Expected Time Complexity: O(n)
## Expected Auxilliary Space: O(1)

## Constraints:

- 0 <= number of nodes <= 10^5
- 0 <= node->data , x <= 10^3
