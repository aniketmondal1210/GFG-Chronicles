# Check if Linked List Length is Even

## Problem Statement

Given the head of a linked list, determine whether the **length of the list is even or odd**.

Return:
- `true` → if length is even  
- `false` → if length is odd  

---

## Examples

### Example 1

**Input**

12 -> 52 -> 10 -> 47 -> 95 -> 0


**Output**

true


**Explanation**  
Length = 6 (even)

---

### Example 2

**Input**

9 -> 4 -> 3


**Output**

false


**Explanation**  
Length = 3 (odd)

---

## Constraints

- 1 <= number of nodes <= 10^5
- 1 <= elements of the linked list <= 10^5
