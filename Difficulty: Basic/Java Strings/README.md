# Concatenate Two Strings

## Problem Statement
You are given two strings `S1` and `S2`. Your task is to concatenate both strings and return the resulting string.

---

## Examples

### Example 1:
**Input:**  
S1 = `"Geeksfor"`  
S2 = `"Geeks"`  
**Output:**  
`GeeksforGeeks`  
**Explanation:**  
Combining `"Geeksfor"` and `"Geeks"` results in `"GeeksforGeeks"`.

---

### Example 2:
**Input:**  
S1 = `"Practice"`  
S2 = `"Hard"`  
**Output:**  
`PracticeHard`  
**Explanation:**  
Combining `"Practice"` and `"Hard"` results in `"PracticeHard"`.

---

## Constraints
- `1 <= |S1|, |S2| <= 10⁵`  
- `|S|` denotes the length of the string `S`.

---

## Expected Time Complexity

`O(|S1| + |S2|)`

## Expected Auxiliary Space

`O(|S1| + |S2|)`
