# Return the Last Digit of a Number

## Problem Statement

Given an integer `n`, write a program to return the **last digit** of the number.

---

## Examples

### Example 1
**Input:**  
`n = 10`  
**Output:**  
`0`

### Example 2  
**Input:**  
`n = 9768`  
**Output:**  
`8`

---

## Constraints

- `-10^9 ≤ n ≤ 10^9`
