# Length of a Singly Linked List

## Problem Statement

Given the head of a singly linked list, return the **length of the list** (i.e., number of nodes).

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5
```

**Output**
```
5
```

---

### Example 2

**Input**
```
2 -> 4 -> 6 -> 7 -> 5 -> 1 -> 0
```

**Output**
```
7
```

---

## Constraints:

- 1 <= number of nodes <= 4*10^4
- 1 <= node->data <= 10^3
