# Find Length of a String

## Problem Statement

You are given a string `s`. Your task is to **find and return the length** of the string.

---

## Examples

### Example 1:
**Input:**  
`s = "Geeks"`  
**Output:**  
`5`

---

### Example 2:
**Input:**  
`s = "Hello1234"`  
**Output:**  
`9`

---

## Constraints

- `1 <= |s| <= 100000`
