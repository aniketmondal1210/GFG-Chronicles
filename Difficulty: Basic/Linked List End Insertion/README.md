# Insert at End of Linked List

## Problem Statement

Given the **head of a singly linked list** and a value `x`, insert a new node with value `x` at the **end** of the list and return the updated head.

---

# Examples

### Example 1

**Input**
```
Linked List: 1 -> 2 -> 3 -> 4 -> 5
x = 6
```

**Output**
```
1 -> 2 -> 3 -> 4 -> 5 -> 6
```

---

### Example 2

**Input**
```
Linked List: 4 -> 5
x = 1
```

**Output**
```
4 -> 5 -> 1
```

---

## Constraints:

- 0 ≤ number of nodes ≤ 10^5
- 0 ≤ node->data , x ≤ 10^3
