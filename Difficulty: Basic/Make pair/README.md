# Form a Pair from Two Elements

## Problem Statement
You are given two elements `x` and `y`.  
The task is to return the **pair** formed with these two elements.

---

## Examples
**Input:**  

x = 4, y = 5

**Output:**  

(4,5)


**Input:**  

x = 1, y = 0

**Output:**  

(1,0)


---

## Constraints
- `1 <= x, y <= 1000`
