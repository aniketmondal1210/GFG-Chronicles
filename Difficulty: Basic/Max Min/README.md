# Sum of Minimum and Maximum Element in an Array

## Problem Statement
You are given an array `A` of size `N`.  
Your task is to **find the sum of the minimum and maximum element** in the array.

---

## Examples

### Example 1
**Input**

N = 5
A[] = {-2, 1, -4, 5, 3}

**Output**

1

**Explanation**  
- Minimum = -4  
- Maximum = 5  
- Sum = -4 + 5 = 1  

---

### Example 2
**Input**

N = 4
A[] = {1, 3, 4, 1}

**Output**

5

**Explanation**  
- Minimum = 1  
- Maximum = 4  
- Sum = 1 + 4 = 5  

---

## Constraints
- `1 <= N <= 10^5`  
- `-10^9 <= Ai <= 10^9`

---

## Expected Complexity
- **Time Complexity**: `O(N)`  
- **Auxiliary Space**: `O(1)`
