# Mean of an Unsorted Array

## Problem Statement

Given an **unsorted integer array `arr[]`**, your task is to **find the mean** of the array.  
**Note:** Return the **floor value** of the mean.

---

## Examples

### Example 1:
**Input:**  
`arr = [1, 3, 4, 2, 6, 5, 8, 7]`  
**Output:**  
`4`  

**Explanation:**  
Sum = 1 + 3 + 4 + 2 + 6 + 5 + 8 + 7 = 36  
Mean = 36 / 8 = 4.5  
Floor of 4.5 is **4**

---

### Example 2:
**Input:**  
`arr = [4, 4, 4, 4, 4]`  
**Output:**  
`4`  

**Explanation:**  
Sum = 4 + 4 + 4 + 4 + 4 = 20  
Mean = 20 / 5 = 4

---

## Constraints

- `1 ≤ arr.length ≤ 10^6`  
- `1 ≤ arr[i] ≤ 10^6`
