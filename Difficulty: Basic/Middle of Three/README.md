# Find Middle of Three Numbers

## Problem Statement

Given three **distinct** numbers `a`, `b`, and `c`, find the number that lies **in between** the other two (i.e., neither minimum nor maximum).

---

# Examples

### Example 1

**Input**
```
a = 978, b = 518, c = 300
```

**Output**
```
518
```

---

### Example 2

**Input**
```
a = 162, b = 934, c = 200
```

**Output**
```
200
```

---

### Example 3

**Input**
```
a = 246, b = 214, c = 450
```

**Output**
```
246
```

---

## Constraints:

- 1<=a, b, c<=10^9
- a, b, c are distinct.
