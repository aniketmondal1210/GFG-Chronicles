# Check if a String is a Pangram

## Problem Statement
You are given a string `s`.  
Your task is to determine whether the string is a **pangram** or not.

A **pangram** is a string that contains **all 26 letters of the English alphabet at least once**, regardless of case.

---

## Examples

### Example 1
**Input:**  

s = "Thequickbrownfoxjumpsoverthelazydog"

**Output:**  

True


**Explanation:**  
The string contains every letter from `'a'` to `'z'` at least once.

---

### Example 2
**Input:**  

s = "HeavyDuty"

**Output:**  

False


**Explanation:**  
Not all letters of the English alphabet are present.

---

## Constraints

1 ≤ |s| ≤ 10⁴
