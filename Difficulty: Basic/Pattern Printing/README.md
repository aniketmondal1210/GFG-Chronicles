# Asterisk Series Pattern

## Problem Statement

Given an integer `N`, print a series of asterisks (`*`) from **1 to N terms** where the number of asterisks increases by **1 in each term**.

Each group of asterisks should be separated by a **space**.

---

## Examples

### Example 1

**Input**
```
N = 3
```

**Output**
```
* ** ***
```

**Explanation**

- First print `1` asterisk  
- Then print `2` asterisks  
- Then print `3` asterisks  

---

### Example 2

**Input**
```
N = 5
```

**Output**
```
* ** *** **** ***** 
```

**Explanation**

The pattern prints increasing asterisks:

```
1 star
2 stars
3 stars
4 stars
5 stars
```

---

## Expected Time Complexity: O(N^2)
---
## Expected Auxiliary Space: O(1)
---
## Constraints:

- 1 <= N <= 1000
