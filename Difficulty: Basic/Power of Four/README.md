# Check if a Number is a Power of Four

## Problem
Given a number `N`, determine if it is a power of 4.  
Return:
- `1` if it is a power of 4
- `0` otherwise.

---

## Examples

### Example 1
**Input:**

N = 64

**Output:**

1

**Explanation:**  
4³ = 64 → N is a power of 4.

---

### Example 2
**Input:**

N = 75

**Output:**

0

**Explanation:**  
75 is not a power of 4.

---

## Constraints
- 1 ≤ N ≤ 10⁵

---

## Expected Complexity
- **Time Complexity:** O(N)
- **Space Complexity:** O(1)
