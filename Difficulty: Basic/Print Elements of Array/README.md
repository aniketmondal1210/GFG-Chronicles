# Print Array Elements (Space-Separated)

## Problem Statement

Given an array `arr[]`, print all its elements space-separated.

📝 **Note:**  
- Do **not** print a new line after printing all elements.  
- Elements should be printed **space-separated**.

---

## Examples

### Example 1

**Input:**  
`arr[] = [1, 2, 3, 4, 5]`  
**Output:**  
`1 2 3 4 5`

---

### Example 2

**Input:**  
`arr[] = [2, 3, 5, 5]`  
**Output:**  
`2 3 5 5`

---

## Constraints

- `1 ≤ arr.size ≤ 10^6`  
- `1 ≤ arr[i] ≤ 10^8`
