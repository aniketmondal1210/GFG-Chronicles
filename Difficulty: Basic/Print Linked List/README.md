# Print a Singly Linked List

## Problem Statement

Given the **head of a singly linked list**, print all elements of the list.

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5
```

**Output**
```
1 2 3 4 5
```

---

### Example 2

**Input**
```
8 -> 1
```

**Output**
```
8 1
```

---

## Constraints:

- 1 ≤ list.size ≤ 10^5
- 0 ≤ node->data ≤ 10^3
