# Print Two Strings with Space

## Problem Statement

Given two inputs stored in variables `a` and `b`, print them in a **single line** with a **space** separating them. Ensure that the output ends with a **newline**.

---

## Examples

### Example 1:
**Input:**  
`a = "Hello"`  
`b = "World"`  
**Output:**  
`Hello World`  

**Explanation:**  
Both `a` and `b` are printed on the same line, separated by a space.

---
