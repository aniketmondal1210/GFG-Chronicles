# K-th Digit from Right of a^b

## Problem Statement
Given two numbers `a` and `b`, find the **k-th digit from the right** in the value of `a^b`.

---

## Examples

### Example 1
**Input:**  
`a = 3, b = 3, k = 1`  
**Output:**  
`7`  

**Explanation:**  
3³ = 27 → 1st digit from right = 7

---

### Example 2
**Input:**  
`a = 5, b = 2, k = 2`  
**Output:**  
`2`  

**Explanation:**  
5² = 25 → 2nd digit from right = 2

---

## Constraints
- `1 ≤ a, b ≤ 15`
- `1 ≤ k ≤ digits in a^b`
