# Count Number of Bit Flips

## Problem Statement

Given two integers `a` and `b`, find the number of bits that need to be flipped to convert `a` into `b`.

👉 Flipping means:
```
0 → 1  OR  1 → 0
```

---

# Examples

### Example 1

**Input**
```
a = 10, b = 20
```

**Output**
```
4
```

---

### Example 2

**Input**
```
a = 20, b = 25
```

**Output**
```
3
```

---

## Constraints: 

- 1 ≤ a, b ≤ 10^6
