# Remove All Characters Except Alphabets

You are given a string `S`.  
Your task is to **remove all characters except English alphabets (A–Z, a–z)**.

- If the resulting string contains **no alphabets**, return `"-1"`.
- Otherwise, return the filtered string.

---

## Examples

### Example 1

Input:
S = "$Gee*k;s..fo, r'Ge^eks?"

Output:
GeeksforGeeks


**Explanation:**  
All non-alphabet characters are removed.

---

### Example 2

Input:
S = "{{{}}> & ^%)"

Output:
-1


**Explanation:**  
No alphabetic characters are present in the string.

---

###   Constraints:

- 1 <= |S| <= 10^5
