# Remove Non-Numeric Characters from a String

## Problem Statement
You are given an **alphanumeric string** `s`.  
Your task is to **remove all characters except numeric characters (0–9)** and return the resulting string.

---

## Examples

### Example 1
**Input:**  

s = "AA1d23cBB4"


**Output:**  

"1234"


**Explanation:**  
All non-numeric characters are removed, leaving only digits.

---

### Example 2
**Input:**  

s = "a1b2c3"


**Output:**  

"123"


**Explanation:**  
Only numeric characters are kept.

---

## Constraints

1 ≤ |s| ≤ 10⁵
