# Remove Consecutive Repeated Digits

## Problem

Given an integer `N` represented as a **string**, remove all **consecutive repeated digits** and return the modified string.

---

## Examples

### Example 1

**Input**

```text
N = "1224"
```

**Output**

```text
"124"
```

**Explanation**

The consecutive `2`s are reduced to a single `2`.

---

### Example 2

**Input**

```text
N = "1242"
```

**Output**

```text
"1242"
```

**Explanation**

There are no consecutive repeated digits.

---

## Your Task:
You don't need to read input or print anything. Your task is to complete the function modify() which takes the integer N as input and returns the modified integer by removing all the consecutive duplicate occurrences in N.


## Expected Time Complexity: O(LogN).
## Expected Auxiliary Space: O(1).


## Constraints:

- 1<=N<=10^18
