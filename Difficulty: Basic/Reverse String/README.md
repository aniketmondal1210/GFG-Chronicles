# Reverse String
Given a string `s`, you need to reverse it.

### Example 1
**Input:**  
`s = "Hello"`  
**Output:**  
`"olleH"`  
**Explanation:**  
Reverse of "Hello" is "olleH".

### Example 2
**Input:**  
`s = "World"`  
**Output:**  
`"dlroW"`  
**Explanation:**  
Reverse of "World" is "dlroW".
