# Reverse String Without Altering Space Positions

## Problem

Given a string `s`, reverse the string **without changing the positions of the spaces**.

---

## Examples

### Example 1

**Input**

```text
s = "Help others"
```

**Output**

```text
"sreh topleH"
```

**Explanation**

The space remains at the same index, while all non-space characters are reversed.

---

### Example 2

**Input**

```text
s = "geeksforgeeks"
```

**Output**

```text
"skeegrofskeeg"
```

**Explanation**

Since there are no spaces, the entire string is reversed.

---

## Constraints:

- 1 ≤ |s| ≤ 10^5
