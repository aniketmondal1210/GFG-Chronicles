# Reverse Only Vowels in a String

## Problem Statement

Given a string `s` consisting of lowercase English letters, reverse **only the vowels** in the string.

👉 Consonants remain in their original positions.

---

# Examples

### Example 1

**Input**
```
s = "geeksforgeeks"
```

**Output**
```
geeksforgeeks
```

---

### Example 2

**Input**
```
s = "practice"
```

**Output**
```
prectica
```

---

### Example 3

**Input**
```
s = "bcdfg"
```

**Output**
```
bcdfg
```

---

## Constraints:

- 1<=|s|<=10^5
