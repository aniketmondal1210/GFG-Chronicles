# Search Element in Linked List

## Problem Statement

Given a **singly linked list** and an integer `x`, check whether `x` is present in the list.

Return:
```
true  → if found
false → otherwise
```

(Driver prints `1` for true, `0` for false)

---

# Examples

### Example 1

**Input**
```
LinkedList: 1 → 2 → 3 → 4 → 5
x = 4
```

**Output**
```
1
```

---

### Example 2

**Input**
```
LinkedList: 2 → 4 → 6 → 7 → 5 → 1 → 0
x = 10
```

**Output**
```
0
```

---

## Constraints:

- 1 <= n <= 100
- 1 <= x, value <= 10^3
