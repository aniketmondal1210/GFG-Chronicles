# Check if an Element Exists in a Vector (Using STL)

---

## Problem Statement
Given a vector `v[]` and an integer `x`, check whether `x` exists in the vector using the STL function `find()`.

Return:
- `true` if `x` exists  
- `false` otherwise  

---

## Examples

**Input:**  
v = [1, 2, 3, 4], x = 4  
**Output:**  
true  

**Input:**  
v = [4, 55, 22], x = 6  
**Output:**  
false
