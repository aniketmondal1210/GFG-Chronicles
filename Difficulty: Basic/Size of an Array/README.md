# Find Size of an Array

## Problem Statement
You are given an array `arr[]`. Your task is to find the **size of the array**.

---

## Constraints
- 1 ≤ |arr| ≤ 1000  

---

## Examples

### Example 1
**Input**

arr[] = [1, 2, 3]

**Output**

3

**Explanation**: The size of the array is 3.

---

### Example 2
**Input**

arr[] = [1, 2, 3, 5, 4]

**Output**

5

**Explanation**: The size of the array is 5.
