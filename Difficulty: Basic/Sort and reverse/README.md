# Sort and Reverse Array

## Problem
Given an array `arr[]` containing non-negative integers,  
the task is to sort the array in **non-decreasing order**,  
and then **reverse it** to get the final result.

---

## Examples

### Input

arr[] = [3, 24, 1, 5]

### Output

[24, 5, 3, 1]


---

### Input

arr[] = [85, 35, 24]

### Output

[85, 35, 24]


---

## Constraints
- `1 <= arr.size() <= 100`  
- `1 <= arr[i] <= 1000`
