# Sum of Elements in an Array

## Problem Statement

You are given an integer array `arr[]`. The task is to **find the sum of its elements** and return the result.

---

## Examples

### Example 1
**Input:**  
`arr[] = [1, 2, 3, 4]`  
**Output:**  
`10`  
**Explanation:**  
`1 + 2 + 3 + 4 = 10`

---

### Example 2
**Input:**  
`arr[] = [1, 3, 3]`  
**Output:**  
`7`  
**Explanation:**  
`1 + 3 + 3 = 7`

---

## Constraints

- `1 <= arr.length <= 10^5`
- `1 <= arr[i] <= 10^4`
