# Sum of Natural Numbers

## Problem Statement

Given an integer `n`, compute the **sum of all natural numbers** from **1 to n** (inclusive).  
If `n` is `0`, the sum should be `0`.

---

## Examples

### Example 1:

**Input:**  
`n = 3`  
**Output:**  
`6`  
**Explanation:**  
1 + 2 + 3 = 6

---

### Example 2:

**Input:**  
`n = 5`  
**Output:**  
`15`  
**Explanation:**  
1 + 2 + 3 + 4 + 5 = 15

---

## Constraints

- `1 ≤ n ≤ 10^4`
