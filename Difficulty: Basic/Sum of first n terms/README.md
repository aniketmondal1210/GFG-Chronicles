# Sum of Cubes of First N Natural Numbers

## Problem Statement

Given an integer `n`, calculate the sum of the series:  
**1³ + 2³ + 3³ + 4³ + … + n³**

---

## Examples

### Example 1:
**Input:**  
`n = 5`  
**Output:**  
`225`  
**Explanation:**  
1³ + 2³ + 3³ + 4³ + 5³ = 1 + 8 + 27 + 64 + 125 = **225**

---

### Example 2:
**Input:**  
`n = 7`  
**Output:**  
`784`  
**Explanation:**  
1³ + 2³ + 3³ + 4³ + 5³ + 6³ + 7³ = 784

---

## Constraints

- `1 ≤ n ≤ 200`
