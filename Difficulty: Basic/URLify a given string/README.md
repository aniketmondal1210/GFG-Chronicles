# Replace Spaces with '%20'

## Problem Statement

Given a string `s`, replace all spaces `' '` with `"%20"`.

---

# Examples

### Example 1

**Input**
```
s = "i love programming"
```

**Output**
```
"i%20love%20programming"
```

---

### Example 2

**Input**
```
s = "Mr Benedict Cumberbatch"
```

**Output**
```
"Mr%20Benedict%20Cumberbatch"
```

---

## Constraints:

- 1 ≤ s.size() ≤ 10^4
