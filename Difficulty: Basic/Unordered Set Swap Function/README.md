# Swap Two Sets

## Problem Statement
You are given **two sets** `st1` and `st2`.  
The task is to **swap** these two sets with each other.

---

## Example
**Input:**  

st1 = ["abc", "bcd", "gfg"]
st2 = ["bmw", "gfg"]


**Output:**  

st1 = ["bmw", "gfg"]
st2 = ["abc", "bcd", "gfg"]


**Explanation:**  
The two sets are swapped directly.
