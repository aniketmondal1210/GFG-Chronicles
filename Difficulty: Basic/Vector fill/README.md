# Fill Vector with Zeroes using STL `fill()`

## Problem Statement
You are given a vector `arr[]`. The task is to fill the vector with zeroes using the **STL function `fill()`**.

---

## Examples

**Input:**  

arr[] = [1, 2, 3, 4]

**Output:**  

[0, 0, 0, 0]


---

**Input:**  

arr[] = [3, 2, 1]

**Output:**  

[0, 0, 0]


---

## Constraints
- `1 ≤ arr.size() ≤ 100`
