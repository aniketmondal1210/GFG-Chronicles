# Check if a Character is a Vowel

## Problem Statement
Given an English alphabet character `c`, determine whether it is a **vowel** or not.

Vowels in English are:

a, e, i, o, u

(Both lowercase and uppercase are considered vowels.)

Return:
- `"YES"` if `c` is a vowel
- `"NO"` otherwise

---

## Example 1

**Input:**

c = 'a'


**Output:**

YES


**Explanation:**
`'a'` is a vowel.

---

## Example 2

**Input:**

c = 'Z'


**Output:**

NO


**Explanation:**
`'Z'` is not a vowel.

---

## Constraints

- c belongs to English alphabets
- c may be lowercase or uppercase


---

## Expected Time Complexity

O(1)


## Expected Auxiliary Space

O(1)
