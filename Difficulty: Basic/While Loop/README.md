# Print Numbers from x to 0 in Decreasing Order

## Problem Statement

Given a number `x`, the task is to **print the numbers from `x` to `0` in decreasing order** in a single line.

---

## Examples

### Example 1:

**Input:**  
`x = 3`  

**Output:**  
`3 2 1 0`  

**Explanation:**  
Numbers in decreasing order from 3 are: `3 2 1 0`.

---

### Example 2:

**Input:**  
`x = 5`  

**Output:**  
`5 4 3 2 1 0`
