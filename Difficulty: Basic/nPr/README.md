# Calculate nPr

## Problem
Write a program to calculate `nPr`.  
`nPr` represents **n permutation r**, and is defined as:

```bash
nPr = n!/(n-r)!
```

---

## Examples

### Example 1
**Input:**

n = 5, r = 2

**Output:**

20

**Explanation:**

5! / (5 - 2)! = 5! / 3! = 120 / 6 = 20


---

### Example 2
**Input:**

n = 6, r = 3

**Output:**

120

**Explanation:**

6! / (6 - 3)! = 6! / 3! = 720 / 6 = 120


---

## Constraints
- 1 ≤ n, r ≤ 20
