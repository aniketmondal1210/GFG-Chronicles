# Add Integer, Float, and Long Using `auto`

## Problem Statement
You are given:
- One integer `a`  
- One float `b`  
- One long `c`  

You need to add all these variables and store the result into an `auto` variable `ans`. Finally, print the result.

---

## Examples

### Example 1
**Input:**  

a = 3, b = 4.2, c = 23

**Output:**  

30.2

**Explanation:**  
`a + b + c = 3 + 4.2 + 23 = 30.2`

---

### Example 2
**Input:**  

a = 2, b = 7.42, c = 100

**Output:**  

109.42

**Explanation:**  
`a + b + c = 2 + 7.42 + 100 = 109.42`

---

## Constraints
- `-10^6 ≤ a ≤ 10^6`  
- `-10^6 ≤ b ≤ 10^6`  
- `-10^9 ≤ c ≤ 10^9`
