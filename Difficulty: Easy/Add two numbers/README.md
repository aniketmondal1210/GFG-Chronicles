# Add Two Numbers using Lambda Expression

## Problem Statement
You need to use a **lambda expression** to generate a function `add(x, y)` that returns the sum of two numbers.

---

## Examples

**Input:**  

x = 5, y = 6

**Output:**  

11


**Input:**  

x = 6, y = 20

**Output:**  

26


---

## Constraints
- `1 <= x, y <= 10^5`
