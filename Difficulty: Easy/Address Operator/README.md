# Print Variable Using Its Address

## Problem Statement
You are given a variable `a`. Your task is to print the value of the variable **through its address** using pointers.

---

## Examples

### Input

a = 4

### Output

4


---

### Input

a = 61

### Output

61


---

## Constraints
- `1 ≤ a ≤ 10^6`
