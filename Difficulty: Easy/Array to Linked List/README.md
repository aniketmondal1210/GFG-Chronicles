# Convert Array to Singly Linked List

## Problem Statement

Given an array `arr[]`, construct a **singly linked list** using its elements and return the **head** of the list.

---

# Examples

### Example 1

**Input**
```
arr = [1, 2, 3, 4, 5]
```

**Output**
```
1 -> 2 -> 3 -> 4 -> 5
```

---

### Example 2

**Input**
```
arr = [10, 11, 12, 13, 14]
```

**Output**
```
10 -> 11 -> 12 -> 13 -> 14
```

---

## Constraints:

- 1 ≤ arr.size() ≤ 10^5
- 1 ≤ arr[i] ≤ 10^5
