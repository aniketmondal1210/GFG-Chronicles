# Convert Binary to Decimal

## Problem Statement
Given a **Binary Number** `n`, find its decimal equivalent.  

---

## Examples

### Example 1
**Input:**  

n = 10001000

**Output:**  

136


---

### Example 2
**Input:**  

n = 101100

**Output:**  

44


---

## Constraints
- `1 <= number of bits in binary number <= 8`
