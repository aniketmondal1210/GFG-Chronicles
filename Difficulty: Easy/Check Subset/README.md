# Check if One Set is a Subset of Another

## Problem Statement

Given two sets `a` and `b`, **check whether `a` is a subset of `b`**.

A set `a` is a **subset** of set `b` if **all elements of `a` are present in `b`**.

---

## Example:

### Example 1:

**Input:**  
`a = [1, 4, 3]`, `b = [1, 4, 3, 2]`  

**Output:**  
`True`
