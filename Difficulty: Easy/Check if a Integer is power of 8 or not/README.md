# Power of Eight

## Problem Statement

Given a positive integer `N`, determine whether it is a power of `8` or not.

Return:
- `"Yes"` if `N` is a power of `8`
- `"No"` otherwise

---

# Examples

### Example 1
```text
Input: 64
Output: Yes
```

### Explanation
```text
64 = 8 × 8 = 8^2
```

So, 64 is a power of 8.

---

### Example 2
```text
Input: 75
Output: No
```

### Explanation
```text
75 cannot be written as 8^x
```

So, it is not a power of 8.

---

# Constraints

```text
1 <= N <= 10^18
```
