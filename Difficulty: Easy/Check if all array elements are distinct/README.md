# Check if All Array Elements Are Distinct

## Problem

Given an array `arr[]` of size `n`, determine whether **all elements are distinct**.

Return:

- `true` if every element appears exactly once.
- `false` if any element appears more than once.

---

## Examples

### Example 1

**Input**

```text
arr = [1, 2, 3, 4, 7]
```

**Output**

```text
true
```

**Explanation**

All elements are unique.

---

### Example 2

**Input**

```text
arr = [1, 3, 2, 4, 5, 1]
```

**Output**

```text
false
```

**Explanation**

The element `1` appears twice, so the array does not contain distinct elements.

---

## Constraints:

- 1 ≤ arr.size() ≤ 10^6
- 1 ≤ arr[i] ≤ 10^9
