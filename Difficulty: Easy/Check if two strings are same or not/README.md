# Compare Two Strings (Case Sensitive)

## Problem Statement

Given two strings `s1` and `s2`, check whether they are exactly the same considering case sensitivity.

Return:

- `true` if both strings are identical
- `false` otherwise

---

# Example 1

```text
Input:
s1 = "abc"
s2 = "abc"

Output:
true
```

---

# Example 2

```text
Input:
s1 = " "
s2 = " "

Output:
true
```

---

# Example 3

```text
Input:
s1 = "geeks"
s2 = "Geeks"

Output:
false
```

### Explanation

```text
'g' and 'G' are different characters because comparison is case-sensitive.
```

---

## Constraints:

- 1 ≤ |s1|, |s2| ≤ 10^5
