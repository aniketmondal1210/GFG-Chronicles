# Print Hexadecimal Value of an Integer

## Problem
You are given an integer `a`.  
Your task is to print its **hexadecimal value**.

**Note:** Move the cursor to the next line after printing the value.

---

## Examples

**Input:**  
a = 26  

**Output:**  

1a


**Input:**  
a = 20  

**Output:**  

14


---

## Constraints
- 1 <= a <= 100
