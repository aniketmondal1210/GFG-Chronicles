# Count Total Number of 1's in a Sorted Binary Array

## Problem Statement
You are given a **binary array** sorted in **non-increasing order** (all the 1's come before all the 0's).  
Your task is to **count the total number of 1's** in the array.

---

## Example 1
**Input:**

arr = [1, 1, 1, 1, 1, 0, 0, 0]


**Output:**

5


**Explanation:**  
The array has five 1's.

---

## Example 2
**Input:**

arr = [1, 1, 1, 1, 1, 1, 1]


**Output:**

7


**Explanation:**  
The array has seven 1's.

---

## Constraints

1 ≤ arr.size() ≤ 10⁵
0 ≤ arr[i] ≤ 1
