# Count Digits in a Number

## Problem Statement

Given an integer `n`, find the **number of digits** in it.

---

# Examples

### Example 1

**Input**
```
n = 1
```

**Output**
```
1
```

---

### Example 2

**Input**
```
n = 99999
```

**Output**
```
5
```

---

## Constraints:

- 1 ≤ n ≤ 10^9
