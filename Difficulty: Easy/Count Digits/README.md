# Count Digits in a Number

## Problem Statement

Given a natural number `n`, find and return the **number of digits** in the number.

---

## Examples

### Example 1

**Input**
```
n = 12
```

**Output**
```
2
```

**Explanation**

The number `12` has **2 digits**.

---

### Example 2

**Input**
```
n = 456
```

**Output**
```
3
```

**Explanation**

The number `456` has **3 digits**.

---

## Constraints:

- 1 ≤ n ≤ 10^5
