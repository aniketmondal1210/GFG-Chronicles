# Count Non-Repeated Elements in an Array

## Problem Statement

Given an array of integers `arr[]`, return the **count of non-repeated elements** in the array.

A **non-repeated element** is an element whose **frequency is exactly 1**.

---

## Examples

### Example 1

**Input**
```
arr[] = [1, 1, 2, 2, 3, 3, 4, 5, 6, 7]
```

**Output**
```
4
```

**Explanation**

Elements with frequency `1`:
```
4, 5, 6, 7
```

Total non-repeated elements = **4**

---

### Example 2

**Input**
```
arr[] = [10, 20, 30, 40, 10]
```

**Output**
```
3
```

**Explanation**

Elements with frequency `1`:
```
20, 30, 40
```

Total non-repeated elements = **3**
