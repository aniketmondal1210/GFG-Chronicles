# Lexicographical Sorting using Counting Sort

## Problem Statement

Given a string **s** consisting of **lowercase English letters**, sort it in **lexicographical (alphabetical) order** using **Counting Sort**.

---

# Examples

### Example 1

**Input**
```
s = "edsab"
```

**Output**
```
abdes
```

---

### Example 2

**Input**
```
s = "geeksforgeeks"
```

**Output**
```
eeeefggkkorss
```

---

## Constraints:

- 1 ≤ s.size() ≤ 10^5
