# Decimal to Binary Conversion

## Problem
Given a decimal number `n`, return its binary equivalent as a string without leading zeros.

---

## Examples

### Example 1
**Input:**

n = 12

**Output:**

1100

**Explanation:**

12 = 1×2³ + 1×2² + 0×2¹ + 0×2⁰


---

### Example 2
**Input:**

n = 33

**Output:**

100001

**Explanation:**

33 = 1×2⁵ + 0×2⁴ + 0×2³ + 0×2² + 0×2¹ + 1×2⁰


---

## Constraints
- 1 ≤ n ≤ 2³¹ - 1
