# Delete Alternate Nodes in Linked List

## Problem Statement

Given a singly linked list:

👉 Delete all nodes at **even positions** (2nd, 4th, 6th, ...)

👉 Keep nodes at **odd positions** (1st, 3rd, 5th, ...)

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5 -> 6
```

**Output**
```
1 -> 3 -> 5
```

---

### Example 2

**Input**
```
99 -> 59 -> 42 -> 20
```

**Output**
```
99 -> 42
```

---

## Expected Time Complexity: O(n)
## Expected Auxiliary Space: O(1)

## Constraints:

- 1 <= number of nodes <= 10^5
- 1 <= node->data <= 10^3
