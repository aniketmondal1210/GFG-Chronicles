# Delete Head of Linked List

## Problem Statement

Given a **Linked List**, delete the **head node** and return the **new head**.

Also:
```
Set the original head to NULL
```

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 1 -> 7
```

**Output**
```
2 -> 3 -> 1 -> 7
```

---

### Example 2

**Input**
```
10 -> 5 -> 7 -> 8 -> 99 -> 100
```

**Output**
```
5 -> 7 -> 8 -> 99 -> 100
```

---

## Constraints:

- 1 ≤ number of nodes ≤ 10^5
- 1 ≤ node->data ≤ 10^5
