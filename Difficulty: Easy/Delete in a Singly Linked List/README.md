# Delete Xth Node from Linked List

## Problem Statement

Given the head of a singly linked list and an integer `x` (1-based index), delete the **xth node** from the list and return the updated head.

---

# Examples

### Example 1

**Input**
```
x = 1
List: 1 -> 2 -> 3 -> 1 -> 7
```

**Output**
```
2 -> 3 -> 1 -> 7
```

---

### Example 2

**Input**
```
x = 5
List: 2 -> 3 -> 4 -> 5 -> 6
```

**Output**
```
2 -> 3 -> 4 -> 5
```

---

## Constraints:

- 1 ≤ size of linked list ≤ 10^5
- 1 ≤ x ≤ size of linked list
