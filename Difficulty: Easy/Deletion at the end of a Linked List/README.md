# Delete Tail of Linked List

## Problem Statement

Given the head of a singly linked list, delete the **last node (tail)** and return the updated head.

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5
```

**Output**
```
1 -> 2 -> 3 -> 4
```

---

### Example 2

**Input**
```
3 -> 12 -> 45
```

**Output**
```
3 -> 12
```

---

## Constraints:

- 1 ≤ number of nodes ≤ 10^5
- 1 ≤ node->data ≤ 10^5
