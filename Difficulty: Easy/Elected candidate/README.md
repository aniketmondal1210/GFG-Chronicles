# Candidate with Maximum Votes

## Problem Statement

Given an array **arr[]** of candidate names (strings), where each name represents a vote, find the **candidate with the maximum votes**.

---

# Examples

### Example 1

**Input**
```
arr = ["Shrikant", "Sushma", "Rajnath", "Sushma"]
```

**Votes Count**
```
Shrikant → 1
Sushma   → 2
Rajnath  → 1
```

**Output**
```
Sushma
```

---

### Example 2

**Input**
```
arr = ["Atal", "Shashtri", "Atal"]
```

**Votes Count**
```
Atal      → 2
Shashtri  → 1
```

**Output**
```
Atal
```

---

## Constraints:

- 1 <= arr.size() <= 10^4
