# Count Distinct Elements in Array

## Problem Statement

Given an integer array `arr[]`, return the **count of all the distinct elements** in the array.

---

## Examples

### Example 1:
**Input:**  
`arr = [2, 2, 3, 2]`  
**Output:**  
`2`  
**Explanation:**  
Distinct elements are {2, 3}

---

### Example 2:
**Input:**  
`arr = [12, 1, 14, 3, 16]`  
**Output:**  
`5`  
**Explanation:**  
All elements are distinct.

---

### Example 3:
**Input:**  
`arr = [1, 1, 1, 1]`  
**Output:**  
`1`  
**Explanation:**  
Only one distinct element {1}

---

## Constraints

- `1 ≤ arr.size() ≤ 10⁵`  
- `-1000 ≤ arr[i] ≤ 1000`
