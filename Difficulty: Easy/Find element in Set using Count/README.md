# Check if an Element Exists in a Set using STL `count()`

## Problem
You are given a set `s` and a number `x`.  
The task is to check whether the number `x` exists in the set or not using the STL function `count()`.

---

### Examples

**Input:**

s = [1, 2, 3, 4], x = 4

**Output:**

true


**Input:**

s = [3, 2, 1], x = 10

**Output:**

false


---

### Constraints
- `1 ≤ s.size() ≤ 100`
