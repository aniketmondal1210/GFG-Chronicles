# Sum of Last N Nodes in Linked List

## Problem Statement

Given a singly linked list and an integer `n`, find the **sum of the last `n` nodes**.

> It is guaranteed that `n` is less than or equal to the number of nodes.

---

# Examples

### Example 1

**Input**
```
Linked List: 5 -> 9 -> 6 -> 3 -> 4 -> 10
n = 3
```

**Output**
```
17
```

---

### Example 2

**Input**
```
Linked List: 1 -> 2
n = 2
```

**Output**
```
3
```

## Constraints:

- 1 <= number of nodes, n <= 10^5
- 1 <= node->data <= 10^3
