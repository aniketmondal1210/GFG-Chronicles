# Find the First Digit of a Number

## Problem Statement

Given a number `n`, **find the first digit** of the number.

---

## Examples

### Example 1:

**Input:**  
`n = 123`  

**Output:**  
`1`

---

### Example 2:

**Input:**  
`n = 976`  

**Output:**  
`9`

---

## Constraints:

- 1 ≤ n ≤ 10⁹
