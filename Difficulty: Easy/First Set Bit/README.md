# Position of First Set Bit (From Right)

## Problem Statement

Given an integer `n`:

👉 Find the position of the **first set bit (1)** from the **right (LSB)**  
👉 Position is **1-based**

👉 If no set bit exists → return `0`

---

# Examples

### Example 1
```
Input:  n = 18
Binary: 10010
Output: 2
```

---

### Example 2
```
Input:  n = 12
Binary: 1100
Output: 3
```

---

### Example 3
```
Input:  n = 1
Binary: 1
Output: 1
```

---

## Constraints:

- 1 ≤ n ≤ 10^9
