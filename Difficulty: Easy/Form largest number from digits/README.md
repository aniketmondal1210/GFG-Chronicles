# Form the Maximum Number from Array Digits

## Problem Statement
You are given an array `arr[]` consisting of digits from **0 to 9**.  
Your task is to **rearrange the elements of the array** such that, after combining all the digits, the **maximum possible number** is formed.

Return the resulting number.

---

## Examples

### Example 1
**Input:**

arr[] = [9, 0, 1, 3, 0]


**Output:**

93100


**Explanation:**
Rearranging the digits in descending order forms the largest possible number: **93100**.

---

### Example 2
**Input:**

arr[] = [1, 2, 3]


**Output:**

321


**Explanation:**
Arranging the digits in descending order gives the maximum number **321**.

---

## Constraints

- 1 <= arr.size() <= 10^6
- 0 <= arr[i] <= 9
