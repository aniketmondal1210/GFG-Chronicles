# Extract Username from Email String

## Problem Statement
You are given a string `s` in **email format**.  
Your task is to take input and print the substring of the string **till the '@' character**.

---

## Examples

**Input:**  

s = hello_world@xyz.com

**Output:**  

hello_world


**Input:**  

s = geeksforgeeks@gfg.com

**Output:**  

geeksforgeeks


---

## Constraints
- `1 <= |s| <= 1000`
- String `s` contains exactly one `'@'`.
