# Insert into Sorted Linked List

## Problem Statement

You are given:
- A **sorted linked list** (ascending order)
- An integer `key`

👉 Insert `key` into the list such that it **remains sorted**

---

# Examples

### Example 1

**Input**
```
25 -> 36 -> 47 -> 58 -> 69 -> 80
key = 19
```

**Output**
```
19 -> 25 -> 36 -> 47 -> 58 -> 69 -> 80
```

---

### Example 2

**Input**
```
50 -> 100
key = 75
```

**Output**
```
50 -> 75 -> 100
```

---

## Constraints:

- 1 <= size of linked list <= 10^6
- 1 <= data of nodes <= 10^6
