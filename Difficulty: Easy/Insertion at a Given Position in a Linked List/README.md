# Insert Node at Given Position (Linked List)

## Problem Statement

You are given:

- Head of a singly linked list
- An integer `pos` (1-based index)
- A value `val`

👉 Insert a new node with value `val` at position `pos` and return the updated head.

---

# Examples

### Example 1

**Input**
```
List: 1 -> 3
pos = 3, val = 4
```

**Output**
```
1 -> 3 -> 4
```

---

### Example 2

**Input**
```
List: 1 -> 2 -> 9
pos = 2, val = 5
```

**Output**
```
1 -> 5 -> 2 -> 9
```

---

## Constraints:

1 ≤ List Size ≤ 10^4 
1 ≤ pos ≤ List Size + 1
1 ≤ val ≤ 10^4
