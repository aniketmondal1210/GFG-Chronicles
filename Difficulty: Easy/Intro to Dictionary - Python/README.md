# Create a Dictionary from Names and Marks

## Problem Statement
You are given:
- An integer `N` — the number of students  
- A list `names` containing **N unique student names**  
- A list `marks` containing **N corresponding marks**

Your task is to create and return a **dictionary** where:
- Key = student name  
- Value = student's marks  

The final output should print the dictionary contents in **sorted order of names**.

---

## Example

### Input:

N = 5
names = [john, ala, ilia, sudan, mercy]
marks = [100, 200, 150, 80, 300]


### Output:

ala 200

ilia 150

john 100

mercy 300

sudan 80


---

## Constraints

- 1 <= N <= 10⁴
- 1 <= marks[i] <= 10⁴
