# Check if Linked List is Sorted

## Problem Statement

Given the head of a singly linked list, determine whether it is **sorted**.

Sorting can be:
- **Non-decreasing (ascending)** OR
- **Non-increasing (descending)**

---

# Examples

### Example 1

**Input**
```
5 -> 5 -> 6 -> 7 -> 8
```

**Output**
```
true
```

---

### Example 2

**Input**
```
2 -> 5 -> 7 -> 8 -> 99 -> 7
```

**Output**
```
false
```

---

## Constraints:

- 1 ≤ list.size ≤ 10^3
- 1 ≤ node->data ≤ 10^3
