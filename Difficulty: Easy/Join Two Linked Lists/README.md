# Append Two Linked Lists

## Problem Statement

Given two singly linked lists:

```
head1 → first list
head2 → second list
```

Append the second list at the end of the first list and return the head of the new list.

---

# Example

### Input
```
List1: 1 → 2 → 3
List2: 4 → 5
```

### Output
```
1 → 2 → 3 → 4 → 5
```

---

## Constraints:

- 1 ≤ number of nodes  ≤ 10^5
- 1 ≤  value of nodes ≤ 10^5
