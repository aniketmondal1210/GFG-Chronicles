# Kth Node From End of Linked List

## Problem Statement

Given the head of a singly linked list and an integer `k`, return the **kth node from the end**.

If `k` is greater than the number of nodes, return `-1`.

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7 -> 8 -> 9, k = 2
```

**Output**
```
8
```

---

### Example 2

**Input**
```
10 -> 5 -> 100 -> 5, k = 5
```

**Output**
```
-1
```

---

## Constraints:

1 ≤ number of nodes ≤ 10^6
1 ≤ node->data , x ≤ 10^6
1 ≤ k ≤ 10^6
