# Square of a Number using Lambda Expression

## Problem Statement
You need to calculate the **square** of a given number `n` using a **lambda expression** in C++.

---

## Examples

**Input:**  

n = 5

**Output:**  

25


**Input:**  

n = 6

**Output:**  

36


---

## Constraints
- `1 <= n <= 10^5`
