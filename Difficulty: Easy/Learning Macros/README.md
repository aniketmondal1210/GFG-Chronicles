# C++ Macro Multiplication

## Problem Statement

**Macros** are an important part of **C++ preprocessors**.  
They allow defining constants or functions that are **replaced during the preprocessing stage** before compilation.

You are given two integers:

```
a and b
```

For each test case, print:

```
a * b
```

using a **macro**.

---

## Input Format

```
T
a b
a b
...
```

- `T` → Number of test cases  
- For each test case → integers `a` and `b`

---

## Output Format

For each test case print:

```
a * b
```

---

## Constraints

```
1 ≤ T ≤ 10
1 ≤ a, b ≤ 100
```

---

## Example

### Input
```
2
5 6
7 11
```

### Output
```
30
77
```
