# Find Lexicographically Smallest and Largest String

## Problem

Given an array of strings `s[]`, find:

- The **lexicographically smallest** string.
- The **lexicographically largest** string.

Return both strings.

---

## Examples

### Example 1

**Input**

```text
s[] = ["a", "ab", "abc"]
```

**Output**

```text
["a", "abc"]
```

**Explanation**

- Lexicographically smallest string is `"a"`.
- Lexicographically largest string is `"abc"`.

---

### Example 2

**Input**

```text
s[] = ["abc", "abc"]
```

**Output**

```text
["abc", "abc"]
```

**Explanation**

Both the smallest and largest strings are `"abc"`.

---

## Constraints:

- 1 ≤ s.size() ≤ 10^3
- 1 ≤ s[i].size() ≤ 10^3
