# Delete Node at Given Position in Linked List

## Problem Statement

Given the head of a singly linked list and an integer `pos` (1-based index), delete the node at that position and return the updated head.

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5, pos = 4
```

**Output**
```
1 -> 2 -> 3 -> 5
```

---

### Example 2

**Input**
```
2 -> 5 -> 7 -> 8 -> 99 -> 100, pos = 6
```

**Output**
```
2 -> 5 -> 7 -> 8 -> 99
```

---

## Expected Time Complexity: O(pos).
## Expected Auxiliary Space: O(1).

## Constraints:

- 2 <= number of nodes <= 103
- 1 <= pos <= n
