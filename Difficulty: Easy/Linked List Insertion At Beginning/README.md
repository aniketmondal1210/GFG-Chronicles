# Insert Node at Beginning of Linked List

## Problem Statement

Given the head of a **singly linked list** and a value `x`, insert a new node with value `x` at the **beginning** of the list and return the updated head.

---

# Examples

### Example 1

**Input**
```
x = 1
List: 2 -> 10
```

**Output**
```
1 -> 2 -> 10
```

---

### Example 2

**Input**
```
x = 1
List: 2 -> 3 -> 4 -> 5
```

**Output**
```
1 -> 2 -> 3 -> 4 -> 5
```

---

## Constraints:

- 1 ≤ number of nodes ≤ 10^5
- 1 ≤ node->data, x ≤ 10^3
