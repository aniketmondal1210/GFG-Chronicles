# Maximize ∑ arr[i] * i

## Problem Statement

Given an array `arr[]`, you can **rearrange its elements**.

👉 Maximize:
```
∑ arr[i] * i   (i = 0 to n-1)
```

Return the result **modulo (10^9 + 7)**.

---

# Examples

### Example 1

**Input**
```
arr = [5, 3, 2, 4, 1]
```

**Output**
```
40
```

---

### Example 2

**Input**
```
arr = [1, 2, 3]
```

**Output**
```
8
```

---

### Example 3

**Input**
```
arr = [7, 7, 7, 7]
```

**Output**
```
42
```

---

## Constraints:

- 1 ≤ arr.size ≤ 10^5
- 1 ≤ arr[i] ≤ 10^5
