# Find Middle Element of Stack

## Problem Statement

Given a stack `st[]` of size `n`, find the middle element.

### Note

- If the number of elements is odd → return the exact middle.
- If the number of elements is even → return the middle element closer to the bottom.

---

# Examples

### Example 1

```text
Input:
st[] = [1, 2, 3, 4, 5]

Output:
3
```

### Explanation

```text
Middle element is 3
```

---

### Example 2

```text
Input:
st[] = [1, 2, 3, 4]

Output:
2
```

### Explanation

```text
Middle elements are 2 and 3
Choose the one closer to the bottom → 2
```

---

## Constraints:

- 1 ≤ st.size() ≤ 10^5
