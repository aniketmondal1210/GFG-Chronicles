# Find Middle of Linked List

## Problem Statement

Given the head of a singly linked list, return the **value of the middle node**.

- If the number of nodes is **odd** → return the middle node
- If **even** → return the **second middle node**

---

# Examples

### Example 1

**Input**
```
1 -> 2 -> 3 -> 4 -> 5
```

**Output**
```
3
```

---

### Example 2

**Input**
```
2 -> 4 -> 6 -> 7 -> 5 -> 1
```

**Output**
```
7
```

---

## Constraints:

- 1 ≤ no. of nodes ≤ 10^5
- 1 ≤ node->data ≤ 10^5
