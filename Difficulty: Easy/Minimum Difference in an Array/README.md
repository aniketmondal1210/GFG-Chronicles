# Minimum Difference Between Any Pair

## Problem Statement

Given an array `arr[]` of positive integers, find the minimum difference between any pair of elements.

---

# Examples

### Example 1

```text
Input:
arr[] = [2, 4, 5, 9, 7]

Output:
1
```

### Explanation

```text
Difference between 5 and 4 is 1
```

---

### Example 2

```text
Input:
arr[] = [3, 10, 8, 6]

Output:
2
```

### Explanation

```text
Difference between 8 and 6 is 2
```

---

## Constraints:

- 2 ≤ n ≤ 10^5
- 1 ≤ arr[i] ≤ 10^9
