# Find Duplicate and Missing Number

## Problem Statement

You are given an array `arr[]` of size `n` containing numbers from **1 to n**.

👉 One number is:
- **missing**
- One number appears **twice**

Return:
```
[duplicate, missing]
```

---

# Examples

### Example 1

**Input**
```
arr = [2, 2]
```

**Output**
```
[2, 1]
```

---

### Example 2

**Input**
```
arr = [1, 3, 3]
```

**Output**
```
[3, 2]
```

---

### Example 3

**Input**
```
arr = [4, 3, 6, 2, 1, 1]
```

**Output**
```
[1, 5]
```

---

## Constraints:

- 2 ≤ n ≤ 10^6
- 1 ≤ arr[i] ≤ n
