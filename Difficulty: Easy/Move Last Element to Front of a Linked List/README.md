# Move Last Node to Front (Linked List)

## Problem Statement

Given the head of a **singly linked list**, move the **last node** to the **front** and return the new head.

---

# Examples

### Example 1

**Input**
```
2 → 5 → 6 → 2 → 1
```

**Output**
```
1 → 2 → 5 → 6 → 2
```

---

### Example 2

**Input**
```
2
```

**Output**
```
2
```

---

## Expected Time Complexity: O(n)
## Expected Space Complexity: O(1)

## Constraints:

- 1 <= size of linked list <= 10^6
- 0 <= elements of list <= 10^9
