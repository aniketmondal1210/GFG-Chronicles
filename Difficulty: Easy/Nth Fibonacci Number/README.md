# Nth Fibonacci Number

## Problem Statement

Given a non-negative integer `n`, return the `nth` Fibonacci number.

The Fibonacci sequence is defined as:

F(0) = 0  
F(1) = 1  
F(n) = F(n - 1) + F(n - 2), for n > 1  

Sequence:

0, 1, 1, 2, 3, 5, 8, 13, 21 ...


---

## Example 1

**Input:**

n = 5


**Output:**

5


---

## Example 2

**Input:**

n = 0


**Output:**

0


---

## Example 3

**Input:**

n = 1


**Output:**

1


---

## Constraints

0 ≤ n ≤ 30
