# Check if Binary Representation is Palindrome

## Problem Statement

Given an integer `n`:

👉 Convert it to binary  
👉 Check if the binary representation is a **palindrome**

---

## Definition

A binary number is a palindrome if:

```
reads same from left → right
and right → left
```

---

# Examples

### Example 1
```
Input:  n = 17
Binary: 10001
Output: true
```

---

### Example 2
```
Input:  n = 16
Binary: 10000
Output: false
```

---

## Constraints:

- 1 ≤ n ≤ 10^9
