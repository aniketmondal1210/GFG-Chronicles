# Palindrome Numbers in a Range

## Problem

Given two integers `m` and `n`, find all palindrome numbers in the range:

```text
[m, n]
```

(inclusive).

A number is a palindrome if it reads the same forward and backward.

---

## Example 1

### Input

```text
m = 10
n = 115
```

### Output

```text
[11, 22, 33, 44, 55, 66, 77, 88, 99, 101, 111]
```

### Explanation

These numbers remain the same when their digits are reversed.

---

## Example 2

### Input

```text
m = 2
n = 5
```

### Output

```text
[2, 3, 4, 5]
```

### Explanation

All single-digit numbers are palindromes.

---

## Constraints:

- 1 ≤ m ≤ n ≤ 10^5
