# Check if String is Palindrome

## Problem Statement

Given a string `s`, determine whether it is a **palindrome**.

A string is called a palindrome if it reads the same **forward** and **backward**.

---

# Examples

### Example 1

**Input**
```
s = "abba"
```

**Output**
```
true
```

---

### Example 2

**Input**
```
s = "abc"
```

**Output**
```
false
```

---

## Constraints:

- 1 ≤ s.size() ≤ 10^6
- The string s contains only lowercase english letters (a-z).
