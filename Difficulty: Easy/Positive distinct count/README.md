# Count Distinct Positive Values

## Problem

Given an array `arr[]`, count the number of **distinct values greater than 0**.

---

## Examples

### Example 1

**Input**

```text
arr = [-3, -2, 0, 3, 4, 5]
```

**Output**

```text
3
```

**Explanation**

Distinct positive values are:

```text
3, 4, 5
```

Count = `3`.

---

### Example 2

**Input**

```text
arr = [-1, -1, -1, -1, 0, 1, 1, 1, 1]
```

**Output**

```text
1
```

**Explanation**

Distinct positive values are:

```text
1
```

Count = `1`.

---

## Constraints:

- 1 ≤ arr.size() ≤ 10^6
- -10^6 ≤ arr[i] ≤ 10^6
