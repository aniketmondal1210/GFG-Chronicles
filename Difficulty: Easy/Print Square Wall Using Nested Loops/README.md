# Print Square Wall Using Nested Loops (Without String Multiplication)

## Problem Statement

Given an integer `n`, write a program to **print a square wall of size `n Ã— n`** using:
- **Nested loops**
- **Avoid using string multiplication**

---

## Examples

**Explanation:**
