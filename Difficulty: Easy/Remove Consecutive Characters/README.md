# Remove Consecutive Duplicate Characters

## Problem Statement

Given a string `s`:

👉 Remove **consecutive duplicate characters**

👉 Keep only **one occurrence** of each consecutive group

---

# Examples

### Example 1
```
Input:  "aabb"
Output: "ab"
```

---

### Example 2
```
Input:  "aabaa"
Output: "aba"
```

---

### Example 3
```
Input:  "abcddcba"
Output: "abcdcba"
```

---

## Constraints:

- 1 ≤ n ≤ 10^6
- All characters in the string are lowercase English alphabets.
