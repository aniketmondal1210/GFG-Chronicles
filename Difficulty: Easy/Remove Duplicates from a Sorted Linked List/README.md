# Remove Duplicates from Sorted Linked List

## Problem

Given the head of a **sorted singly linked list**, remove all duplicate nodes so that each element appears **only once**.

The linked list should remain sorted.

**Note:** Solve the problem without using extra space.

---

## Examples

### Example 1

**Input**

```text
Head: 2 -> 2 -> 4 -> 5
```

**Output**

```text
2 -> 4 -> 5
```

**Explanation**

The value `2` appears twice, so one occurrence is removed.

---

### Example 2

**Input**

```text
Head: 2 -> 2 -> 2 -> 2 -> 2
```

**Output**

```text
2
```

**Explanation**

All duplicate occurrences of `2` are removed, leaving only one node.

---

## Constraints:

- 1 ≤ Number of nodes, data of nodes ≤ 10^5
