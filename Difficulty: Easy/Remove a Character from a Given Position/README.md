# Remove Character at a Given Position

## Problem

Given a string `s` and a **0-based index** `pos`, remove the character present at that position and return the modified string.

---

## Example 1

### Input

```text
s = "abcde"
pos = 1
```

### Output

```text
"acde"
```

### Explanation

Character at index `1` is:

```text
'b'
```

After removing it:

```text
"acde"
```

---

## Example 2

### Input

```text
s = "a"
pos = 0
```

### Output

```text
""
```

### Explanation

The only character is removed, resulting in an empty string.

---

## Constraints:

- 1 ≤ |s| ≤ 2 * 10^5
