# Remove All Occurrences of a Character from String

## Problem Statement

Given a string `s` and a character `c`, remove all occurrences of `c` from the string.

---

# Example 1

```text
Input:
s = "geeksforgeeks"
c = 'e'

Output:
"gksforgks"
```

---

# Example 2

```text
Input:
s = "geeksforgeeks"
c = 'g'

Output:
"eeksforeeks"
```

---

## Constraints:

- 1 <= |s| <= 10^5
