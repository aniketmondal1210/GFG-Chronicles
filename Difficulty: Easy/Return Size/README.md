# Store Size of Data Types in C++

## Problem Statement

Given some variables in C++, you need to store the **size (in bytes)** of the following data types into three integer variables:

- `a` → size of `int`
- `b` → size of `double`
- `c` → size of `char`

You only need to store the sizes into `a`, `b`, and `c`.

---

## Example

### Input
```
None
```

### Output
```
4 8 1
```

### Explanation

Typical sizes (may vary by system):

- `int` → 4 bytes  
- `double` → 8 bytes  
- `char` → 1 byte
