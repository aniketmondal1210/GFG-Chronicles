# Reverse String Using Stack

## Problem Statement

Given a string `s`, reverse the string using a stack.

---

# Examples

### Example 1

```text
Input:
s = "GeeksforGeeks"

Output:
"skeeGrofskeeG"
```

---

### Example 2

```text
Input:
s = "Geek"

Output:
"keeG"
```

---

## Constraints:

- 1 ≤ s.length() ≤ 100
