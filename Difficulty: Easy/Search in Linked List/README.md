# Search Key in Linked List

## Problem Statement

Given the **head of a linked list** and an integer `key`, check whether the key is present in the linked list.

Return:
```
true  → if key exists  
false → otherwise
```

---

# Examples

### Example 1

**Input**
```
Linked List: 1 → 2 → 3 → 4
key = 3
```

**Output**
```
true
```

---

### Example 2

**Input**
```
Linked List: 1 → 2 → 3
key = 4
```

**Output**
```
false
```

---

## Constraint:

- 1 ≤ Number of nodes ≤ 10^5
- 1 ≤ node.data, key ≤ 10^5

---

## Expected Complexities:

- Time Complexity: O(n)
- Auxiliary Space: O(1)
