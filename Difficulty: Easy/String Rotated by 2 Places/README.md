# String Rotation by Exactly 2 Places

## Problem Statement

Given two strings `s1` and `s2`:

👉 Check if `s2` can be obtained by rotating `s1`  
👉 Rotation must be **exactly 2 places**  
👉 Direction must be **same (either left OR right)**

---

# Examples

### Example 1
```
Input:  s1 = "amazon", s2 = "azonam"
Output: true
```

---

### Example 2
```
Input:  s1 = "geeksforgeeks", s2 = "geeksgeeksfor"
Output: false
```

---

### Example 3
```
Input:  s1 = "ab", s2 = "ab"
Output: true
```

---

## Constraints:

- 1 ≤ s1.length, s2.length ≤ 10^5
