# Sum of All Array Elements

## Problem Statement

Given an integer array `arr[]`, return the **sum of all elements** in the array.

---

## Examples

### Example 1

**Input**
```
arr[] = [54, 43, 2, 1, 5]
```

**Output**
```
105
```

**Explanation**
```
54 + 43 + 2 + 1 + 5 = 105
```

---

### Example 2

**Input**
```
arr[] = [324, 5, 2, 2]
```

**Output**
```
333
```

**Explanation**
```
324 + 5 + 2 + 2 = 333
```
