# Sum of Digits of a Number

## Problem Statement

Given a positive number `n`, find the **sum of all the digits** of `n`.

---

## Examples

### Example 1:
**Input:**  
`n = 687`  
**Output:**  
`21`  
**Explanation:** `6 + 8 + 7 = 21`

---

### Example 2:
**Input:**  
`n = 12`  
**Output:**  
`3`  
**Explanation:** `1 + 2 = 3`

---

## Constraints

- `1 ≤ n ≤ 10⁵`
