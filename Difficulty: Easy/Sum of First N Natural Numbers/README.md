# Sum of First N Natural Numbers

## Problem Statement

Given an integer `n`, compute the sum of all natural numbers from **1 to n** (inclusive).  
If `n` is `0`, the sum should be `0`.

---

## Examples

### Example 1:
**Input:**  
`n = 6`  
**Output:**  
`21`  
**Explanation:**  
1 + 2 + 3 + 4 + 5 + 6 = 21

---

### Example 2:
**Input:**  
`n = 4`  
**Output:**  
`10`  
**Explanation:**  
1 + 2 + 3 + 4 = 10

---

### Example 3:
**Input:**  
`n = 0`  
**Output:**  
`0`  
**Explanation:**  
Since `n = 0`, the sum is `0`.

---

## Constraints

- `0 ≤ n ≤ 10⁴`
