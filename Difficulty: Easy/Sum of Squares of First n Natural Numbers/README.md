# Sum of Squares of First N Natural Numbers

## Problem Statement

Given an integer `n`, find the sum of squares of the first `n` natural numbers.

```text
1² + 2² + 3² + ... + n²
```

---

# Example 1

```text
Input:
n = 2

Output:
5
```

### Explanation

```text
1² + 2²
= 1 + 4
= 5
```

---

# Example 2

```text
Input:
n = 3

Output:
14
```

### Explanation

```text
1² + 2² + 3²
= 1 + 4 + 9
= 14
```

## Constraints:

- 0 <= n <= 10^3
