# Swap Two Numbers

## Problem
You are given two integers `a` and `b`.  
The task is to **swap** the two numbers.  
Swapping means interchanging the values of `a` and `b`.

---

## Examples

### Example 1
**Input**

a = 3, b = 5

**Output**

5 3

**Explanation**  
Initially, `a = 3` and `b = 5`. After swapping â†’ `a = 5`, `b = 3`.

---

## Constraints
- `1 <= a, b <= 10^6`
