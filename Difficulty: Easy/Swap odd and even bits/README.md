# Swap All Even and Odd Bits

## Problem Statement

Given an unsigned integer `n`:

👉 Swap every:
- even-position bit
with its adjacent
- odd-position bit

---

# Examples

### Example 1
```
Input:  n = 23

Binary:
00010111

After swapping:
00101011

Output: 43
```

---

### Example 2
```
Input:  n = 2

Binary:
10

After swapping:
01

Output: 1
```

---

## Constraints:

- 1 ≤ n ≤ 10^9
