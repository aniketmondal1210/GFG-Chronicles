# Swapping Objects using Classes in C++

## Problem Statement
You are given two objects that each have three attributes:
- `name`
- `number`
- `height`

Your task is to **swap the values of two objects** using **classes and objects** in C++.

---

## Example 1
**Input:**  

obj1 → Morty 1234 6
obj2 → Rick 5678 5


**Output:**  

Rick 5678 5
Morty 1234 6
