# Typecast Double to Integer

## Problem Statement

Given a double value `d`, typecast it to an integer value and print it.

---

## Example

### Example 1
**Input:**  
`d = 10.23`  
**Output:**  
`10`  
**Explanation:**  
The integer value of 10.23 is 10.

---

## Constraints

- The input `d` is a floating-point number (double precision)
