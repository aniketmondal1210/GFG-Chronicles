# Sum of Elements in a Vector using `accumulate()`

## Problem Statement
You are given a vector `arr[]`.  
The task is to find the **sum of all elements** in the vector using the STL function `accumulate()`.

---

## Examples
**Input:**  

arr[] = [1, 2, 3, 4]

**Output:**  

10


**Input:**  

arr[] = [3, 2, 1]

**Output:**  

6


---

## Constraints
- `1 ≤ arr.size() ≤ 10^5`  
- `1 ≤ arr[i] ≤ 10^4`
