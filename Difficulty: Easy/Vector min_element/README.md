# Find Minimum Element in Vector using STL

## Problem
Given a vector `arr[]`. The task is to find the **minimum element** in the vector using the STL function `min_element()`.

---

### Examples

**Input:**

arr[] = [1, 2, 3, 4]

**Output:**

1


**Input:**

arr[] = [3, 2, 1]

**Output:**

1


**Input:**

arr[] = [5, 4, 2, 8, 8]

**Output:**

2


---

### Constraints
- `1 <= arr.size() <= 100`
