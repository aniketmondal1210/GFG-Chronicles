# Delete Given Node (Without Head)

## Problem Statement

You are given a pointer to a node `del_node` in a singly linked list.

👉 You must delete this node **without access to the head** of the list.

---

# Important Constraint

```
The given node is NOT the last node
```

---

# Examples

### Example 1

**Input**
```
1 -> 2, del_node = 1
```

**Output**
```
2
```

---

### Example 2

**Input**
```
10 -> 20 -> 4 -> 30, del_node = 20
```

**Output**
```
10 -> 4 -> 30
```

---

## Constraints:

- 2 ≤ number of nodes ≤ 10^6  
- 1 ≤ node->data ≤ 10^6
