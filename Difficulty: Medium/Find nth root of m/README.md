# Nth Root of M

## Problem Statement

Given two integers `n` and `m`, find the `n-th` root of `m`.

Return:
- the integer root if it exists
- `-1` otherwise

---

# Examples

### Example 1

```text
Input: n = 3, m = 8
Output: 2
```

### Explanation

```text
2^3 = 8
```

So, the 3rd root of 8 is 2.

---

### Example 2

```text
Input: n = 3, m = 9
Output: -1
```

### Explanation

```text
No integer x exists such that x^3 = 9
```

---

### Example 3

```text
Input: n = 4, m = 16
Output: 2
```

### Explanation

```text
2^4 = 16
```

---

# Constraints

```text
1 ≤ n ≤ 9
0 ≤ m ≤ 10^9
```
