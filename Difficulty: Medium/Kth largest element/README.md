# Kth Largest Element in an Array

## Problem Statement

Given an array **arr[]** and an integer **k**, find the **kth largest element** in the array.

---

# Examples

### Example 1

**Input**
```
arr = [3, 5, 4, 2, 9]
k = 3
```

**Sorted Array (Descending)**

```
[9, 5, 4, 3, 2]
```

The **3rd largest element** is:

```
4
```

**Output**
```
4
```

---

### Example 2

**Input**
```
arr = [4, 3, 7, 6, 5]
k = 5
```

**Sorted Array (Descending)**

```
[7, 6, 5, 4, 3]
```

The **5th largest element** is:

```
3
```

**Output**
```
3
```

---

## Constraints:

- 1 <= k <= arr.size<= 10^6
- -10^4 <= arr[i] <= 10^4
