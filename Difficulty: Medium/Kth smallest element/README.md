# Kth Smallest Element in an Array

## Problem Statement

Given an array **arr[]** containing **distinct positive integers** and an integer **k**, find the **kth smallest element** in the array.

---

# Examples

### Example 1

**Input**
```
arr = [3, 5, 4, 2, 9]
k = 3
```

**Sorted Array**

```
[2, 3, 4, 5, 9]
```

The **3rd smallest element** is:

```
4
```

**Output**
```
4
```

---

### Example 2

**Input**
```
arr = [4, 3, 7, 6, 5]
k = 5
```

**Sorted Array**

```
[3, 4, 5, 6, 7]
```

The **5th smallest element** is:

```
7
```

**Output**
```
7
```

---

## Constraints:

- 1 <= arr.size() <= 10^6
- 1 <= arr[i] <= 10^6
- 1 <= k <= arr.size()
