# Count Number of Digit '2' from 0 to N

## Problem Statement

Given an integer `n`, count how many times digit `'2'` appears in all numbers from `0` to `n`.

---

# Example 1

```text
Input:
n = 22

Output:
6
```

### Explanation

```text
Numbers containing digit 2:

2   -> 1
12  -> 1
20  -> 1
21  -> 1
22  -> 2

Total = 6
```

---

# Example 2

```text
Input:
n = 100

Output:
20
```

---

## Constraints:

- 1 <= T <= 100
- 1 <= N <= 10^5
