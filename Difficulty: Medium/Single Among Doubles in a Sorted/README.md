# Find the Single Element in a Sorted Array

You are given a **sorted array** `arr[]` where:
- Every element appears **exactly twice**  
- Except **one element** that appears **only once**

Your task is to return that single element.

---

## Examples

### Example 1

Input: arr = [1, 1, 2, 2, 3, 3, 4, 50, 50, 65, 65]
Output: 4


### Example 2

Input: arr = [5]
Output: 5


### Example 3

Input: arr = [1, 2, 2, 3, 3]
Output: 1


---

### Constraints:
- 1 ≤ arr.size() ≤ 2 * 10^6
- 1 ≤ arr[i] ≤ 10^6
