# Sort Array by Count of Set Bits

## Problem Statement

Given an array `arr[]`:

👉 Sort in **descending order of number of set bits (1s)** in binary form  

👉 If two numbers have same set bits → **maintain original order (stable sort)**

---

# Examples

### Example 1
```
Input:
[5, 2, 3, 9, 4, 6, 7, 15, 32]

Output:
[15, 7, 5, 3, 9, 6, 2, 4, 32]
```

---

### Example 2
```
Input:
[1, 2, 3, 4, 5, 6]

Output:
[3, 5, 6, 1, 2, 4]
```

---

## Constraints:

- 1 ≤ arr.size() ≤ 10^5
- 1 ≤ arr[i] ≤ 10^6
